EdInfoHub

This project is an Educational Resource Website aimed at providing a comprehensive platform for Ghanaian students to access free educational videos, practice exercises, and study materials in various subjects.

Secure Authentication

This web application built with the MERN stack ensures robust security for user authentication.

It implements industry-standard protocols and encryption techniques to protect user credentials and sensitive data.

Top Notch Security

By utilizing features like bcrypt hashing and JSON Web Tokens (JWT), it ensures that only authorized users can access the application and its resources, guarding against unauthorized access and potential security threats.

A Blog

The web application includes a dynamic and user-friendly blog feature that allows users to create, edit, and publish blog posts. Users can engage with the content by leaving comments and sharing posts on social media platforms. The blog feature integrates seamlessly with the MERN stack, providing a reliable and scalable platform for content creation, management, and user interaction.

A Simple Text Editor

This web application, powered by the MERN stack, offers a straightforward and intuitive text editing feature. Users can create, modify, and format text content directly within the application.

It also supports basic text manipulation operations such as cut, copy, and paste. With its user-friendly interface and responsive design, the simple text editor empowers users to effortlessly compose and edit text content within the application.

Various Options

The editor provides essential functionalities like text styling, formatting options (e.g., bold, italic, underline), and paragraph alignment.

About This Project

I started learning Web development because i wanted to use it to help my students even after they have moved on to other levels of education, we've been having chats over the years and looking at the current state of our educational system in Ghana, i came up with the idea that since our leaders are pushing technology into education but not really doing much, why not create an environment where students can learn from teachers other than those in their schools.

This is why I decided to create an online platform where teachers could upload resources for students to learn. By making use of the internet, I can reach out to a much larger audience and help more people with their education. I am excited to see how far I can go with this initiative and what new opportunities will come out of it.

This is a Portfolio Project for Holberton School Software Engineering course.

Daniel Mbedobe KUNJI
